# python-practice
Python practice for the college

### HW 1 Задания по типам данных, инструкциям, синтаксису языка и исключениям


1. Дополнить фрагмент функционалом: программа запрашивает имя пользователя и выводит его на экран.
2. Дополнить фрагмент функционалом: пользователь вводит число, а на экран выводится квадрат этого числа, корень и факториал (использовать пакет math).
3. Дополнить фрагмент функционалом: пользователь вводит два числа, а на экран выводится сумма, разность, произведение и частное этих чисел.
4. Дополнить фрагмент функционалом: пользователь вводит свой возраст. Программа выводит на экран текст “Ваш возраст делится на 2” только, если возраст пользователя является чётным числом.
5. Дополнить фрагмент функционалом: программа задаёт пользователю любой вопрос и выводит положительный результат, если пользователь ответил верно. В противном случае выводит сообщение об ошибке.
6. Дополнить фрагмент функционалом: пользователь вводит число n. Если n положительное число, то программа выводит сообщение “Привет” n раз.
7. Добавить поддержку исплючений.
8. Написать игру “угадай число”, в которой пользователь будет вводить числа до тех пор, пока не угадает задуманное число. При этом программа будет подсказывать пользователю большее или меньшее число он ввёл. Реализовать вывод подсказок с помощью своих собственных исключений.


### HW 2 Задания по структурам данных

1. Написать фрагмент кода, который принимает список чисел и возвращает сумму этих чисел.
2. Написать фрагмент кода, который принимает список чисел и возвращает максимальный и минимальный элементы.
3. Написать фрагмент кода, который принимает список элементов и возвращает список уникальных элементов.
4. Написать фрагмент кода, который принимает строку и возвращает перевернутую версию этой строки ('hello' → 'olleh').
5. Написать фрагмент кода, который принимает два списка и возвращает список, который состоит из элементов как первого списка, так и второго (['hello', 'hi'], ['hi', 'bye'] → ['hello', 'hi', 'bye']).
6. Написать фрагмент кода, который принимает в качестве аргумента строку и возвращает список слов, из которых состоит строка ('hi hi bye bye bye' → ['hi', 'bye']).
7. Написать фрагмент кода, который принимает в качестве агрумента строку и возвращает словарь. Ключом словаря являются слово, которые встречается в строке, а значением - сколько раз это слово встречается в строке ('hi hi bye bye bye' → {'hi': 2, 'bye': 3}).
8. Написать фрагмент кода, который принимает список элементов и возвращает список только тех элементов, которые встречаются несколько раз в списке.

### HW 3 Задания по функциям и декораторам

1. Напишите функцию, которая осуществляет проверку на то, является ли строка палиндромом. Палиндром — это слово или фраза, которые одинаково читаются слева направо и справа налево.
2. Написать функцию, принимающую 1 аргумент — сторону квадрата, и возвращающую 3 значения (с помощью кортежа): периметр квадрата, площадь квадрата и диагональ квадрата.
3. Пользователь делает вклад в размере amount рублей сроком на years лет под percent % годовых (каждый год размер его вклада увеличивается на percent %. Эти деньги прибавляются к сумме вклада, и на них в следующем году тоже будут проценты). Написать функцию bank, принимающая аргументы amount, years и percent, и возвращающую сумму, которая будет на счету пользователя.
4. Написать декоратор, который будет приводить передаваемый во внутреннюю функцию текст к верхнему регистру.
5. Написать декоратор, который выводит текст внутренней функции несколько раз.
6. Написать кэширующий декоратор, который сохраняет значение функции при каждом вызове. При повторном вызове функции возвращается сохраненное значие
